Formulating an environment of particles with unique behaviours by inducing real world forces

IDEA

01

Playing with code gave a sense of control to color, size and movement. In this project, I wanted to experiment with fades and overlapping of lines to form the structure. Following the same methodology of a basic module and breathing life into it through functions that added character to each module, I created The Weed

The basic idea is the use of a _



1  Module   +  1   Color   +  2  Functions 

PROCESS

02

Each particle that populates the screen has its own properties and scale. It is drawn towards a larger force that maintains the order of the system. Without which the particle can walk randomly without any external control. This code generates a new pattern of Weed for every new runtime as the decision is taken by the particle itself.


The pattern was a result of the traces left behind by the particle in constant motion navigating its way to find new relationships.

Underlying trajectory
ORDER

IN CHAOS



Function 1 : Rotation

The basic module rotates about the pivot point . The direction of rotation is randomized depending on the choice the system makes from the given four choices.

Function 2 : Randomness

Each module moves towards a central force that holds the system together. The force is similar to our Gravitational force of attraction.

CODE

03

Particle[] P1;
PVector centre;
void setup()
{
 size(900,900);
 smooth();
 background(255);
 P1=new Particle[10];
 for(int i=0;i<10;i++)
{
 centre=new PVector(random(-400,400),random(-400,400));
 P1[i]= new Particle(centre);
}

}
void draw()
{
  translate(width/2,height/2);
 for(int i=0;i<10;i++)
 {
   P1[i].step();
   P1[i].check();
   P1[i].checkedge();
   P1[i].update();
   P1[i].show();
 }
}
